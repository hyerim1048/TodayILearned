### sudo su 몇 개의 옵션들 이해하기

```
sudo su [username]
```
* 기본적으로 유저에게 root 권한을 주되, 사용 환경은 그대로 둔다

```
sudo su
```
* 유저에게 root권한을 줌과 동시에 root로 로그인한 것과 같은 환경으로 바뀐다.
