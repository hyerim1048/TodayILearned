### Virtual Adress Space
- every process get 4G virtual address space
- code : 기계어 코드
- data : static, 전역변수를 위해 존재
- heap : 프로그래머의 동적 할당을 위해 존재
- stack : 지역 변수 저장
- page : virtual address space 를 4096bytes 로 나눈 메모리 불록
