## LDAP

- Lightweight Directory Access Protocol
    - Internet protocol that email and other programs use to look up information from a server
- 모두가 접근할 수 있는 phone book 만들기 or 이메일을 한번도 보내지 않은 유저의 주소 찾기 등의 문제를 어떻게 해결할까?
- LDAP servers : index all the data and filter to select person or group the client wat
- contact 뿐만이 아니라 printers, other services on a network 에도 사용됨
- single sign-on : one password for a user is shared between many servives
- appropriate for directory-like information
